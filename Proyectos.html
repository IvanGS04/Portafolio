Detalles del Proyecto
Anterior Siguiente

En este proyecto que elabore en mi estadía de TSU en el CECYTEA plantel Pabellón de Arteaga. Arme y programe los carritos robot con los que contaba el plantel, la programación fue en Arduino con el lenguaje C++. Los carritos tenían la función de moverse hacia en frente, atrás, derecha e izquierda. Tenían un módulo Bluetooth que les permite ser controlados con cualquier dispositovo movil que cuente con funcion Bluetooth. Yo arme los robots desde 0, actualizando el software de cada una de las tablas arduino y tambien la configuración de todos los modulos (Bluetooth y de aproximidad). No cuento con todas las imagenes porque son propiedad de la institución.



Como primer paso era armar completamente el carrito para poder hacer su programación.

Despues se realizo toda la programación en C++ en el ide de arduino.

Detalles del Proyecto
Anterior Siguiente

Este proyecto fue desarrollado en equipo por tres personas. El proyecto consistía en consumir una API por medio de una aplicación de Android, la temática de la aplicación es en improvisar un restaurante llamado KangrePonce, la aplicación pide un inicio de sesión, la cual al registrarse se enviaban todos los datos a la API, al iniciar sesión como admin puede registrar, eliminar y actualizar un cliente, los pedidos y los platillos. Al iniciar sesión como cliente podía hacer pedidos de lo que hubiera dentro del menú. La API fue desarrollada con Javascript y MongoDB, La aplicación fue desarrollada en Android Studio.
